### 🛠 技术栈
- **后端**: ThinkPHP 5 + FastAdmin
- **前端**: 微信小程序 + ColorUI
- **核心插件**: zpwxsys（招聘系统）

### 📋 核心功能
| 模块 | 说明 |
|------|------|
| 👤 用户 | 登录、注册、个人中心 |
| 🏢 企业 | 企业认证、职位发布 |
| 💼 职位 | 全职、兼职、实习 |
| 📝 报名 | 职位报名、报名管理 |
| 📋 任务 | 任务发布、接单 |
| 💬 消息 | 系统消息、留言 |
| ⭐ 评价 | 用户评价、评分 |
| 💰 支付 | 微信支付、钱包 |
| 👥 代理 | 分销、邀请 |

### 🚀 快速开始
1. 后端部署：配置 `addons/zpwxsys/config.php`
2. 小程序配置：修改 `addons/zpwxsys/wxapp/utils/config.js`
3. 微信支付：配置 `addons/zpwxsys/library/WxPay/WxPay.Config.php`
